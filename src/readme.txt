# Vapaat tehtävään liittyvät kommentit?
	Aikaisemmissa ohjelmointi tehtävissä olen saanut miinuspisteitä muun muossa return lauseista, Scalassa voi tehdä
	asioita monella eri tavalla, ja vaan sen takia että funktioita voi rakentaa ilman return lauseita ei ole minun
	mielestäni riittävä hyvää syy pisteiden laskemiseen jos se kuitenkin on tehty johdonmukaisesti. Minun mielestäni
	return lauseet vaan selkeyttävvät koodia.
